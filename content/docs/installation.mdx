# **Installation**

You can get started with Pochi by installing Pochi's VSCode extension.

- **VSCode Marketplace**
  1. **Open VSCode:** Launch the VSCode application.
  2. **Navigate to Extensions:** Click on the Extensions icon in the Activity Bar on the side of VSCode (`Ctrl + Shift + X` or `Cmd + Shift + X`).
  3. **Search for Pochi:** In the Extensions search bar, type `Pochi`.
